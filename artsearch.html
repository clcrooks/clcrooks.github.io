Case Study

Art Search

Designing interfaces for Archival Search using DeepFace

Description

Art Search frames ML and facial recognition as both artifacts and as archaeological tools. Using phenotypic matching the project explores bias in existing computer vision systems like DeepFace, while continuing to craft and source its own archival training data.

Artists
Viv Pitter, Sarah Cheung, Laila Brown, Karina Lopez, Saizami Tripathi, Anna Cho-Son | POIETO PROJECTS x Feminist AI Cohort 2021-2022
Location
Los Angeles
Stack
React Web App, AWS, DeepFace
Role
Designed the interface and worked with developers to bring a team's project vision to life
